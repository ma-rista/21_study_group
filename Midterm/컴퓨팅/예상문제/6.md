# 📘 조건문 예상문제

## 중요 내용
```
1. 제어문 : 명령이 실행되는 순서를 제어하는 명령문
   - 제어구조
   - 조건문(if, if~else, 중첩 if 문, 삼항 연산자)
   - if문 : 조건식 참이면 실행, 거짓이면 프로그램 종료
2. 기본 if문
3. if~else문
4. 중첩 if 문
5. 삼항 연산자
   - 조건식이 참이면 왼쪽의 표현식으로 수행 그렇지 않으면 오른쪽 표현식 수행

```
